# GitHub Project Board Template

Create a GitHub Project with these columns:

| Todo | In Progress | Review | Done |
| --- | --- | --- | --- |
| Add validation tests | Improve Playwright coverage | Review CI setup | Initial appointment system |
| Add deployment target | Add release notes | Review branch protection | Agent guide |
| Add package workflow | Add security docs | Review Dependabot PRs | CodeQL workflow |

## Suggested Labels

- `bug`
- `enhancement`
- `documentation`
- `tests`
- `ci-cd`
- `security`
- `dependencies`
- `frontend`
- `backend`

## Starter Issues

1. Add invalid frontend form Playwright test
2. Add deployment target for demo app
3. Create v1.0.0 release
4. Enable branch protection on `main`
5. Enable secret scanning in repository settings
